# Introduccion
Nginx (pronunciado "engine-x") es un servidor web y un servidor proxy inverso de código abierto que destaca por su rendimiento, escalabilidad y eficiencia. Fue creado por Igor Sysoev y lanzado por primera vez en 2004. Originalmente diseñado para ser un servidor web ligero y de alto rendimiento, Nginx ha ganado popularidad como servidor proxy inverso, equilibrador de carga, y se utiliza comúnmente como servidor para manejar aplicaciones web modernas.

1. Rendimiento y Eficiencia:
Nginx está diseñado para manejar una gran cantidad de conexiones simultáneas con baja utilización de recursos. Su arquitectura asincrónica le permite manejar muchas solicitudes concurrentes de manera eficiente.

2. Servidor Web:
Nginx puede actuar como un servidor web, sirviendo contenido estático de manera rápida y eficiente. Es especialmente adecuado para servir archivos estáticos como HTML, CSS, imágenes y scripts JavaScript.

3. Proxy Inverso:
Uno de los roles más comunes de Nginx es actuar como un servidor proxy inverso. Puede dirigir el tráfico desde los clientes a uno o más servidores backend, lo que facilita la escalabilidad y la distribución de la carga.

4. Equilibrador de Carga:
Nginx puede utilizarse como un equilibrador de carga para distribuir el tráfico entre varios servidores backend. Esto mejora la disponibilidad y la capacidad de respuesta del sistema al distribuir la carga entre los servidores.

5. onfiguración Basada en Bloques:
La configuración de Nginx se realiza a través de archivos de configuración que contienen bloques de configuración. Esto permite una configuración más modular y fácil de mantener.

6. SSL/TLS:
Nginx incluye soporte nativo para SSL/TLS, permitiendo la configuración de conexiones seguras para proteger la transferencia de datos entre el servidor y los clientes.

7. Amplia Adopción:
Nginx es ampliamente utilizado por sitios web populares, servicios en la nube y empresas de todo el mundo debido a su rendimiento, flexibilidad y robustez.

8. Licencia Open Source:
Nginx se distribuye bajo una licencia de software libre BSD, lo que significa que es gratuito para su uso, modificación y distribución.
